# Excel Expense Dashboard

An interactive Expense Dashboard developed in Microsoft Excel using VBA Automation, Pivot Tables, Charts, and AI-generated VBA code.

## Features

- One-click dashboard generation
- Automated Pivot Tables
- Dynamic KPI Cards
- Interactive Slicers
- Expense Analysis
- Monthly Spending Trends
- Category-wise Expense Breakdown
- Professional Dashboard Design

## Tools Used

- Microsoft Excel
- VBA
- Pivot Tables
- Pivot Charts
- Slicers
- Claude AI (for VBA generation)

## Dashboard Insights

- Total Expenses
- Highest Spending Category
- Monthly Expense Trend
- Category-wise Distribution
- Interactive Filtering

## Skills Demonstrated

- Excel Automation
- VBA Programming
- Data Cleaning
- Data Visualization
- Business Intelligence
- Dashboard Development
